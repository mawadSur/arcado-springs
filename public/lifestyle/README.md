# Drop AI lifestyle images here

Save the Higgsfield-generated images in this folder with these exact names. The site picks them up
automatically (no code changes). See `../../HIGGSFIELD_PROMPTS.md` for the prompts.

- `hero.jpg` — 16:9, full-bleed hero (people on the main street). If missing, the hero falls back to `site-plan.jpg`.
- `walking.jpg` — 16:9, people walking the main street
- `dining.jpg` — 4:3, outdoor patio dining
- `shopping.jpg` — 4:3, shoppers along the retail frontage

Tip: keep each file under ~400 KB (JPG quality ~80) for fast loading.
